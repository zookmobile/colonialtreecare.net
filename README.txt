# Colonial Tree Care Website

Official website for Colonial Tree Care, built and maintained by Intuitive Diagnostics LLC.

## Overview
A static website showcasing Colonial Tree Care's services, service area, and contact information.

## Tech Stack
- HTML / CSS / JavaScript
- Hosted via GitHub Pages

## Local Development
1. Clone the repo
2. Open `index.html` in your browser, or serve locally with a tool like `live-server`

## Deployment
Changes pushed to the `main` branch deploy automatically via GitHub Pages.
